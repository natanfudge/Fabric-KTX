Generic utility library for Kotlin Fabric mods

## Gradle

Add `jcenter()` to repositories if you haven't yet:
```groovy
repositories {
    // [...]
    jcenter()
}
```
And add to dependencies:
```groovy
dependencies {
    // [...]
    modImplementation("com.lettuce.fudge:fabric-ktx:$total_version")
}
```
